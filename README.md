Симуляция экосистемы: Зайцы, Волки и Цветы

Визуальная экосистемная симуляция на Java с графическим интерфейсом, демонстрирующая взаимодействие хищников,
травоядных и ресурсов в замкнутой среде.

Что это за проект?
Это интерактивная симуляция, в которой на игровом поле размером 51×51 клетку взаимодействуют три типа объектов:
1) 🐇 Зайцы — травоядные, которые ищут цветы и избегают волков;
2) 🐺 Волки — хищники, которые охотятся на зайцев;
3) 🌸 Цветы — ресурс для зайцев, восстанавливающий энергию.
   
Симуляция показывает, как животные принимают решения на основе ограниченной информации о своём окружении.

🎮 Особенности
Интеллектуальное поведение:

1) Зайцы:
Видят волков в радиусе 2 клеток и убегают от них,
При отсутствии угрозы ищут ближайшие цветы,
При отсутствии цветов двигаются случайно,
Теряют энергию каждый ход,
Собирают цветы для восстановления энергии (+2).

2) Волки:
Видят зайцев в радиусе 2 клеток и преследуют их,
При отсутствии добычи двигаются случайно,
Теряют энергию каждый ход,
При поимке зайца восстанавливают энергию (+2),

Цветовая кодировка:

1) 🟢 Зайцы (H) — светло-зелёные клетки;
2) 🔴 Волки (W) — светло-красные клетки;
3) 🟡 Цветы (F) — светло-жёлтые клетки;
4) ⬜ Пустые клетки — белые.
   
Панель статистики в реальном времени:
1) Количество живых зайцев и волков;
2) Собранные и оставшиеся цветы;
3) Средняя энергия популяций;
4) Текущий ход.

Технологии:
1) Java Swing — графический интерфейс;
2) GridLayout — отображение игрового поля;
3) Многопоточность — автоматическое воспроизведение;
4) Объектно-ориентированный дизайн — классы Hare, Wolf, Flower.

Управление:
1) «Следующий ход» — выполнить один шаг симуляции;
2) Автоматический режим (код содержит закомментированную функцию autoPlay()).
   
📊 Начальные параметры:
Размер поля - 51×51;
1) Зайцы	60 (энергия: 10);
2) Волки	30 (энергия: 10);
3) Цветы	500;
4) Радиус зрения	- 2 клетки;
5) Потеря энергии за ход	1.

🔄 Механика хода:
Каждый ход происходит следующее:

1) Движение зайцев:
Поиск ближайшего волка в радиусе видимости,
Если волк найден → убегание в противоположную сторону,
Если нет → поиск ближайшего цветка,
Если цветок найден → движение к нему,
Если ничего не найдено → случайное движение,
При попадании на цветок → съедание (+2 энергии),
Трата 1 энергии.

2) Движение волков:
Поиск ближайшего зайца в радиусе видимости,
Если заяц найден → преследование,
Если нет → случайное движение,
Трата 1 энергии.

Проверка столкновений:

1) Если волк и заяц на одной клетке → заяц умирает, волк ест (+2 энергии);
2) Если энергия падает до 0 → существо умирает и удаляется с поля.

🎯 Условия окончания игры:
1) Все зайцы вымерли (волки остаются без пищи);
2) Все волки вымерли (зайцы бесконтрольно размножаются).

📈 Статистика в реальном времени:

Программа отображает:
1) Количество живых особей каждого вида;
2) Собранные цветы (всего и осталось);
3) Среднюю энергию популяций;
4) Легенду символов на поле.
